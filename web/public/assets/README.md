# Nerum Brand Assets

Logo final e isotipo para Nerum Finance.

## Archivos

| Archivo | Uso |
|---|---|
| `nerum-logo-horizontal.svg` | Logo principal para tema claro |
| `nerum-logo-horizontal-dark.svg` | Logo principal para tema oscuro |
| `nerum-isotipo.svg` | Isotipo para tema claro |
| `nerum-isotipo-dark.svg` | Isotipo para tema oscuro |
| `nerum-favicon.svg` | Favicon/app icon |
| `nerum-logo-preview.svg` | Lamina de presentacion de variantes |

## Concepto

La marca usa una `N` fluida dentro de un contenedor redondeado. La forma combina claridad, balance y movimiento financiero. El acento mint sugiere crecimiento y acompanamiento sin usar simbolos literales de dinero.

## Colores

| Token | Color |
|---|---:|
| Brand deep | `#0F3D37` |
| Primary | `#168570` |
| Mint | `#A8DCC1` |
| Off-white | `#FAF8F3` |
| Dark bg | `#0E1412` |
| Text light | `#F4F0E8` |

## Uso

- Usar `nerum-logo-horizontal.svg` en headers, login, landing y documentos.
- Usar el isotipo cuando el espacio sea reducido: sidebar colapsada, favicon, mobile nav o avatar de producto.
- En dark mode usar las variantes `dark`.
- No deformar, rotar, cambiar colores ni agregar sombras fuertes al logo.

## Nota tecnica

El wordmark usa `Satoshi` como primera opcion y fallback a `Inter`/system UI. Si se necesita un logo cerrado para imprenta o distribucion externa, conviene convertir el texto a contornos desde Figma/Illustrator.
